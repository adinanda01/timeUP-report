Key Features

Privacy & Security

TimeUp uses smart privacy options (Manifest.v3) with:

  • ‚úì No tracking of sensitive websites
  • ‚úì No access to passwords
  • ‚úì Permissions limited to alarm, storage, notifications, scripting (read-only), and tabs

Real-Time Tracking

The extension updates time usage every 5 seconds.

Browsing Limit Adjustment

Users can set a custom usage time limit for any website.

Smart Alerts

On extended usage of a specific website, TimeUp sends a notification alert to the user.

Reports & Analytics

Daily and weekly reports help users track both productive and non-productive time spent on Chrome.

Additional Key Features

Safe Deleting

To handle memory issues, TimeUp automatically deletes the oldest data when necessary.

Productive Websites Marking

Users can mark websites as productive and receive reward notifications for high usage of those sites. No time-limit alerts will appear for marked productive sites.

Media-Specific Tracking

Implements a hybrid range system: 5-second active tracking followed by 30-minute idle tracking for media websites.

Efficiency & Low Resource Usage

Uses only session storage (not local storage) to maintain lightweight performance.

Customization Options

Idle Detection feature allows tracking even when the user is away, ensuring complete session coverage.
